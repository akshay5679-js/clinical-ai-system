About This Project

The AI Clinical Decision Support System is designed to assist doctors and patients in analyzing medical information using Artificial Intelligence.

Project Modules

  • Heart Disease Risk Prediction using Machine Learning
  • Medical Report Summarization using Large Language Models
  • Medical Question Answering using Retrieval-Augmented Generation (RAG)
  • Explainable AI Dashboard for interpreting predictions

Technology Stack

  • Python
  • Django Web Framework
  • Scikit-learn for Machine Learning
  • LangChain and FAISS for RAG
  • OpenAI API for Medical Summarization

Developer
